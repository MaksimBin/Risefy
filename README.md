🟪 RiSeFY — Music PWA

RiSeFY — прогрессивное веб‑приложение (PWA) для музыки и аудиовизуализации.  
Создано для мобильных экранов: плейлисты, эквалайзер, визуализация и удобный интерфейс — всё в одном.

---

🟪 Основные возможности
- 🟪 Регистрация и авторизация через email  
- 🟪 Плейлисты и коллекции — добавляйте треки с устройства  
- 🟪 Эквалайзер и визуализация — динамичные спектры в реальном времени  
- 🟪 Полноэкранный режим — переключение одним нажатием  
- 🟪 Android PWA — установка как приложение, проверено антивирусом  
- 🟪 Поддержка донатов — встроенная система пожертвований  

---

🟪 Интерфейс
- 🟪 Чёрный фон и минималистичный дизайн  
- 🟪 Закруглённые блоки и плавный скролл  
- 🟪 Фиолетовые акценты для элементов управления  
- 🟪 Визуализация звука в реальном времени  

---

🟪 Установка
1. Откройте сайт https://risefy.org на мобильном устройстве  
2. Установите PWA через браузер (Android поддерживается напрямую)  
3. Для Android доступен APK: скачать

🟪 Технологии
- 🟪 HTML / CSS / JavaScript

- 🟪 Аудио API для визуализации  
- 🟪 UI/UX дизайн для мобильных экранов
